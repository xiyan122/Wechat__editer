# 行为准则（Code of Conduct）

本项目旨在成为一个友好、专业、尊重彼此的协作空间。参与本项目（包括但不限于 Issue、讨论、PR、代码评审、社区沟通）即表示你同意遵守本行为准则。

## 我们的承诺

我们承诺：

- 以尊重与善意对待每一位参与者
- 关注问题本身，避免人身攻击或贬低
- 在分歧中保持专业，优先用事实、复现与数据沟通
- 让贡献过程尽量清晰、可预期（文档化、可复现、可 review）

## 期望的行为

- 友善、建设性、具体：提出可操作的建议，尽量给出复现步骤或示例
- 尊重不同水平与背景：欢迎新手提问，鼓励耐心指导
- 保护隐私与安全：不要公开他人隐私信息、账号信息、未披露的漏洞细节
- 遵循项目约定：按目录分区提交、跑 `npm run build`、尽量聚焦单一主题

## 不可接受的行为

- 侮辱、骚扰、歧视、仇恨言论或任何形式的人身攻击
- 恶意挑衅、刷屏、骚扰评审、强行拉扯无关话题
- 发布或索取敏感个人信息、账号凭据
- 明知有害仍推动合入（例如引入明显不兼容公众号粘贴环境的复杂依赖或外链资源）

## 维护与执行

维护者有权：

- 要求修改不符合规范的内容
- 关闭/锁定讨论、Issue 或 PR
- 在必要时限制或禁止参与

## 举报渠道

如遇到违反行为准则的情况：

- 请在 Issue/讨论中 @ 维护者，或通过仓库主页的联系方式私信维护者（如有）
- 若涉及安全/隐私问题，请尽量私下报告，避免公开扩散

感谢你帮助我们一起维护一个健康的协作环境。
